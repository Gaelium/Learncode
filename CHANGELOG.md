# Changelog

## [1.0.0] - 2026-02-16

### Initial Release

- **EPUB and PDF import** with automatic code extraction and exercise assembly
- **Built-in book reader** — EPUB HTML rendering with KaTeX math support; PDF canvas rendering with page-by-page navigation
- **Exercise sidebar** with chapter/section hierarchy and completion percentages
- **Language detection** for Rust, Python, JavaScript, TypeScript, Go, C/C++, Java, Ruby, and Bash
- **Code block classification** — example, exercise, incremental, output, config, and REPL categories
- **Exercise sandboxes** with language-specific starter templates
- **Progress tracking** — per-exercise status, timestamps, and reset counts persisted in `progress.json`
- **Text annotations** — highlight text in the book reader and add notes, persisted in `annotations.json`
- **Custom worksheets** — coding (multiple languages), SQL, markdown, and blank templates
- **KaTeX math rendering** for inline and display LaTeX notation
- **Reading position memory** — automatically resumes where you left off
